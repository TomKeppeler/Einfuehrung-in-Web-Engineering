Frage:
Gibt es in TypeScript statische Klassen? Erläutern Sie den Unterschied von TypeScript zu anderen objekt-orientierten, statisch typisierten Programmiersprachen wie Java und C#.

Antwort:
Nein, statische Klassen gibt es nicht in Typescript denn es ist nicht nötig. Sprachen wie Java Java brauchen statische Klassen. Daten und Funktionen können ohne einer Klasse nicht existieren. Also gibt es in Java kein Objekt ohne Klasse. Statische Klassen sind also die möglichkeit dennoch, Daten und Funktionen, zu nutzten ohne die Klasse zu instanziieren.
In Java und C# können statische Klassen nur statische Methoden enthalten.
Statische Klassen werden ohne inizialiesirung genutzt werden.
